# Petrungaro Multiservizi

E-commerce italiano for MULTISERVIZI FUNEBRI SRL — funeral ceramics, headstones, and memorial products.

Built with Next.js 16, Stripe, Prisma (PostgreSQL), Resend, Zustand, and Tailwind CSS.

## Setup

```bash
npm install
npx prisma generate
# Copy .env.example to .env and fill values
npm run dev
```

See `AGENTS.md` for full project documentation.
